# 赚钱助手 UI 全面改版设计

## 背景

当前 UI 基于 glassmorphism 风格（毛玻璃 + 紫色渐变），存在以下问题：
- 视觉风格偏花哨，与金融工具的专业感不匹配
- 表格边框线断裂、列间距不一致
- 深浅主题缺乏统一的设计语言
- 信息密度不够，部分页面空间浪费

用户期望对标**同花顺 App** 的设计语言进行改版。

## 设计方向

### 风格定位：同花顺式金融工具

- **品牌色**：蓝色 `#3b6ff5`，替代当前紫色渐变
- **背景**：纯色背景（深色 `#12161f` / 浅色 `#f5f6fa`），去除毛玻璃和径向渐变
- **卡片**：扁平实色卡片（深色 `#1a1f2b` / 浅色 `#ffffff`），轻度描边或投影
- **涨跌色**：保持红涨绿跌（深色 `#e64545`/`#2db860`，浅色 `#cf2e2e`/`#13964b`）
- **圆角**：统一 6-8px，方正利落
- **投影**：极简，仅卡片必要时使用浅投影

### 配色体系

| Token | 深色 | 浅色 |
|-------|------|------|
| 页面背景 | `#12161f` | `#f5f6fa` |
| 卡片/行背景 | `#1a1f2b` | `#ffffff` |
| 侧边栏背景 | 微渐变 `#1a2233→#141c2b` | 微渐变 `#f0f3fa→#e8ebf2` |
| 分割线 | `#1e2748` | `#e8ebf2` |
| 主文字 | `#f4f6ff` | `#1d2740` |
| 次要文字 | `#6b7394` | `#8e96b8` |
| 品牌蓝 | `#3b6ff5` | `#3b6ff5` |
| 涨/红 | `#e64545` | `#cf2e2e` |
| 跌/绿 | `#2db860` | `#13964b` |

### 字体

保持系统字体栈：`'PingFang SC', 'Noto Sans SC', sans-serif`。数字使用 tabular-nums 对齐。

### 布局结构

- **侧边栏**：收窄至 52px，纯图标无文字标签，选中态为品牌蓝实心方块
- **顶部**：大盘指数横条（一行紧凑展示 4 个指数）
- **表头**：汇总行（持仓数 + 总市值 + 总收益）+ 排序切换
- **内容区**：紧凑表格（`border-collapse: collapse`），左侧红/绿色条标识涨跌
- **表格列**：名称/代码 | 现价 | 涨幅 | 市值 | 收益

### 导航

保持 5 个 Tab：自选股票、基金、板块、资产、消息。侧边栏图标 + tooltip 提示。

## 关键改动

1. **CSS 变量体系重写**：9 个旧 token → 新 token 集合
2. **表格统一**：`border-collapse: collapse` + `border-right` 列分隔
3. **侧边栏瘦身**：70px → 52px
4. **去除毛玻璃**：移除 `backdrop-filter`、`blur`、径向渐变背景
5. **详情页面板**：统一返回按钮 + 标题栏样式
